# 梅花易数彩票预测

基于梅花易数算法的彩票号码预测网页应用，一键起卦，同时生成双色球和大乐透号码。

## 功能特点

- 🔮 **一键起卦**：点击按钮即可根据当前时间起卦
- 📊 **同时预测**：一次预测生成双色球和大乐透号码
- 🎨 **中国风界面**：深红、墨黑、金色配色，立体号码球效果
- 📱 **响应式设计**：适配手机和电脑端
- 📖 **卦象解读**：显示本卦、变卦、互卦及体用关系
- 🔍 **详细推导**：点击可查看号码生成的具体计算过程
- 🌟 **气运测试**：测试今日气运，影响号码生成
- 📈 **数据统计**：访问次数、访客人数、起卦次数统计

## 算法说明

### 核心原理

本应用基于《梅花易数》的时间起卦法，结合五行生克关系生成彩票号码。

### 关键概念

- **先天八卦数**：乾1、兑2、离3、震4、巽5、坎6、艮7、坤8
- **体用关系**：体者为主，用者为客，生克比和决定运算方式
  - 体生用/用生体 → 加法，系数调整
  - 体克用/用克体 → 减法（绝对值），系数调整
  - 比和 → 乘法，系数调整
- **归藏**：数有尽时，循环往复，此为变易之道
- **卦变跳跃**：去重时使用先天八卦数作为步长，动爻阴阳决定方向

### 五行生克系数

| 关系 | 系数 | 说明 |
|------|------|------|
| 体生用 | 0.7 | 付出多，数偏小 |
| 体克用 | 1.3 | 主控局面，数偏大 |
| 用生体 | 1.1 | 得助力，数适中偏大 |
| 用克体 | 0.9 | 受制约，数适中偏小 |
| 比和 | 1.0 | 中和，数居中 |

### 工具函数

| 函数 | 易学解释 | 作用 |
|------|---------|------|
| `guiCang(n, max)` | 卦数归藏 | 将任意数字映射到1-max范围 |
| `tongChiPeiShu(candidates, max, dongYao)` | 同池配数（卦变跳跃法） | 同一池内去重排序，步长1-8，方向由动爻决定 |
| `tiYongBianShu(ti, yong, relation, base, max)` | 体用变数 | 根据体用关系和五行系数计算数值 |
| `getWuXingCoeff(relation)` | 五行生克系数 | 获取生克关系对应的系数 |

### 号码生成规则

#### 福彩双色球（6红+1蓝）

**红球池（6个来源）**：
| 序号 | 来源 | 公式 | 归一化 |
|------|------|------|--------|
| ① | 本卦卦序 | `benGuaXu` | `guiCang(benGuaXu, 33)` |
| ② | 变卦卦序 | `bianGuaXu` | `guiCang(bianGuaXu, 33)` |
| ③ | 体用变数 | 体用关系+动爻 | `tiYongBianShu(ti, yong, rel, dong, 33)` |
| ④ | 互卦积 | `huShang × huXia` | `guiCang(积, 33)` |
| ⑤ | 本卦组合 | `上卦×10 + 下卦` | `guiCang(组合, 33)` |
| ⑥ | 变卦×五行系数 | `bianGuaXu × coeff` | `guiCang(乘积, 33)` |

**蓝球**：
```
blueBall = tiYongBianShu(ti, yong, rel, dong+时辰, 16)
```

#### 体彩大乐透（5红+2蓝）

**红球区（5个来源）**：
| 序号 | 来源 | 公式 | 归一化 |
|------|------|------|--------|
| ① | 本卦卦序 | `benGuaXu` | `guiCang(benGuaXu, 35)` |
| ② | 变卦卦序 | `bianGuaXu` | `guiCang(bianGuaXu, 35)` |
| ③ | 体用变数 | 体用关系+动爻 | `tiYongBianShu(ti, yong, rel, dong, 35)` |
| ④ | 互卦积+时辰 | `huShang×huXia + 时辰` | `guiCang(和, 35)` |
| ⑤ | 动爻×时辰×系数 | `dong×时辰×coeff` | `guiCang(乘积, 35)` |

**蓝球区（2个）**：
```
backA = guiCang(huShangNum + dongYao, 12)
backB = guiCang(huXiaNum + diZhi, 12)
```

### 去重算法（卦变跳跃法）

```javascript
function tongChiPeiShu(candidates, maxVal, dongYao) {
    // 步长：先天八卦数（1-8）
    const step = (val % 8) + 1;
    
    // 方向：动爻奇偶决定（阳爻进，阴爻退）
    const direction = dongYao % 2 === 1 ? 1 : -1;
    
    // 跳跃式去重
    v += direction * step;
}
```

**理论依据**：
- 步长使用先天八卦数，符合万物之数本源
- 方向由动爻阴阳决定，体现卦变往来之理
- 跳跃而非连续，避免生成连续数字

## 部署到 GitHub Pages

1. 将项目代码推送到 GitHub 仓库
2. 进入仓库 **Settings** → **Pages**
3. Source 选择 **Deploy from a branch**
4. Branch 选择 **main**，文件夹选择 **/ (root)**
5. 点击 **Save**
6. 等待 1-2 分钟，访问 `https://你的用户名.github.io/仓库名/`
