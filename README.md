# 梅花易数 · 时辰起卦

> 观象于天，取数于时

基于《梅花易数》时间起卦法的双色球与大乐透号码娱乐工具。应用根据北京时间、农历日期和时辰生成卦象，并将结果确定性地映射为两种彩票号码。

在线体验：<https://peiorange888.github.io/meihua-lottery/>

## 功能

- 按北京时间和农历年月日时起卦，展示本卦、变卦、互卦、动爻和体用关系
- 分别生成双色球（6 红 + 1 蓝）与大乐透（5 前区 + 2 后区）
- 同一时辰重复起卦时复用相同结果
- 提供气运解读、预测记录、开奖匹配和历史归档
- 支持桌面和移动端访问

## 算法

时间起卦流程：

1. 读取北京时间，换算农历年支数、农历月、农历日和时辰数。
2. `上卦 = 年支数 + 农历月 + 农历日`，`下卦 = 上卦 + 时辰数`，分别归藏到 8。
3. 总数归藏到 6 得到动爻，再推导变卦、互卦和体用关系。

归藏公式：`归藏(n, max) = n % max === 0 ? max : n % max`。

号码生成使用四个独立号码池：双色球红球 `1–33`、蓝球 `1–16`，大乐透前区 `1–35`、后区 `1–12`。卦象信息作为种子，对每个号码池做确定性洗牌后取所需号码，因此结果可复现且不会超出官方号码范围。

彩票开奖结果是随机事件，本项目的号码仅供学习和娱乐，不代表概率优势或投注建议。

## 本地运行

```bash
python3 -m http.server 8080
```

然后访问 <http://localhost:8080>。

运行测试：

```bash
npm ci
npm test
```

## 部署与安全

项目可直接部署到 GitHub Pages。详细步骤见 [`DEPLOY.md`](DEPLOY.md)。

应用使用 Firebase Realtime Database 保存公开预测和开奖数据，并由 GitHub Actions 定时结算。数据库规则和安全边界见 [`SECURITY.md`](SECURITY.md) 与 [`firebase-database.rules.json`](firebase-database.rules.json)。不要将 Firebase 服务账号或其他私密凭据提交到仓库。
