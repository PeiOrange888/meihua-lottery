# 梅花易数彩票预测

基于梅花易数算法的彩票号码预测网页应用，一键起卦，同时生成双色球和大乐透号码。

## 功能特点

- 🔮 **一键起卦**：点击按钮即可根据当前时间起卦
- 📊 **同时预测**：一次预测生成双色球和大乐透号码
- 🎨 **中国风界面**：深红、墨黑、金色配色，立体号码球效果
- 📱 **响应式设计**：适配手机和电脑端
- 📖 **卦象解读**：显示本卦、变卦、互卦及体用关系
- 🔍 **详细推导**：点击可查看号码生成的具体计算过程

## 算法说明

### 核心原理

本应用基于梅花易数的时间起卦法，结合五行生克关系生成彩票号码。

### 关键概念

- **先天八卦数**：乾1、兑2、离3、震4、巽5、坎6、艮7、坤8
- **体用关系**：体者为主，用者为客，生克比和决定运算方式
  - 体生用/用生体 → 加法
  - 体克用/用克体 → 减法（绝对值）
  - 比和 → 乘法
- **归藏**：数有尽时，循环往复，此为变易之道

### 工具函数

| 函数 | 易学解释 | 作用 |
|------|---------|------|
| `guiCang(n, max)` | 卦数归藏 | 将任意数字映射到1-max范围 |
| `tongChiPeiShu(candidates, max)` | 同池配数 | 同一池内去重排序 |
| `tiYongBianShu(ti, yong, relation, base, max)` | 体用变数 | 根据体用关系选择运算后归藏 |

### 号码生成规则

#### 福彩双色球（6红+1蓝）
- 红球池：1-33，同池内不重复
- 蓝球：1-16，独立计算，可能与红球重复

#### 体彩大乐透（5红+2蓝）
- 红球区：1-35，同池内不重复
- 蓝球区：1-12，同池内不重复，可能与红球区重复

## 使用方法

### 本地运行

直接双击 `index.html` 文件在浏览器中打开即可使用。

### 在线访问

部署到 GitHub Pages 后，可通过以下方式访问：
- 直接访问 GitHub Pages 链接
- 扫描二维码（可在部署后添加）

## 部署到 GitHub Pages

1. 将项目代码推送到 GitHub 仓库
2. 进入仓库 **Settings** → **Pages**
3. Source 选择 **Deploy from a branch**
4. Branch 选择 **main**，文件夹选择 **/ (root)**
5. 点击 **Save**
6. 等待 1-2 分钟，访问 `https://你的用户名.github.io/仓库名/`
